# 云南省企业就业失业数据采集系统

## 项目概述
本项目是云南省企业就业失业数据采集系统的开发仓库，用于实现企业就业失业数据的规范化采集、审核、汇总和上报功能。

## 项目背景
传统人工填报、线下汇总模式存在数据上报不及时、统计口径不统一等痛点，影响人社部门研判与监管实效。为规范数据采集、审核、汇总、上报全流程，实现省、市、企业三级数据互联互通，保障数据真实可追溯，特建设本系统。

## 主要功能
1. **企业备案管理** - 企业用户完善备案信息并提交审核
2. **数据填报** - 企业按月填报就业失业数据
3. **市级审核** - 市级管理员审核辖区企业数据
4. **省级汇总上报** - 省级管理员终审并上报国家部委
5. **数据分析可视化** - 多维度数据分析与报表生成
6. **系统管理** - 用户权限、参数配置、系统监控等
